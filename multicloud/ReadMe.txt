Плюсы и минусы развёртывания магазина sausage-store в multicloud.

Плюсы:
- Легко масштабировать систему, например добавить ещё одного облачного провайдера.
- Большая отказоустойчивость, в случае проблем у одного из провайдеров.
- Легкость доступа и соответствие правовым нормам, если провайдеры расположены в разных юресдикциях.

Минусы: 
- Нужно регистрироваться и платить за обслуживание, а также заключать SLA для нескольких провайдеров.
- Пердётся настраивать взаимодействие сервисов в разных облаках.
- Оба провайдера должны бать выбраны из числа достаточно безопасных.
- Ребликация баз данных между облаками может вызывать задержки.
